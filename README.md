# Compiler Design Assignment


Anyone who wants to add codes for the assignment is free to do so.<br>

## How to contribute.
 - Fork this repository.
 - Clone that repository.
 - Commit and push your code.
 - Then create a pull request.
 
Note: Please notify everyone what question you are working on in the [discussion group](https://github.com/sdsubhajitdas/Compiler-Design-Assignment/issues).

## What to contribute.
Get in the discussion panel under Github Issues and tell what question you want to contribute. [**Discussion Group**](https://github.com/sdsubhajitdas/Compiler-Design-Assignment/issues)
